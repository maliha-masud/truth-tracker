General Information

System Overview

Truth Tracker is an AI-based detection system designed to identify and combat misinformation, fake news, and disinformation across various online platforms.

System Summary

Truth Tracker operates on various devices with internet connectivity. It is accessible through web browsers on desktop computers, laptops, tablets, and smartphones with internet access.

Browser Compatibility: Truth Tracker is compatible with all major web browsers, including Google Chrome, Mozilla Firefox, Safari, Microsoft Edge, and Opera.

Internet Connectivity: The application requires a stable internet connection to perform analysis tasks and retrieve updated information.

Installation: Truth Tracker is a web-based application and does not require installation. Users can access it directly through their web browser by visiting the Truth Tracker website.

Minimum System Requirements: There are no specific minimum system requirements for accessing Truth Tracker. However, a modern web browser and a stable internet connection are recommended for optimal performance.

Usage: After accessing the Truth Tracker website, users can input text snippets for analysis. The results will be displayed directly within the web browser interface.

No Further Configuration Required: Truth Tracker does not require any additional configuration or setup after accessing the website. Users can immediately start using the application to analyze information and identify misinformation.

Data Privacy: Truth Tracker prioritizes user privacy and data security. Personal user data is handled in accordance with privacy regulations and best practices.

User Access Levels

Everyone can use application, but registered users are able to view detailed classification information and track their search history.

Admins are able to view search history and site traffic.

App Functionality

1. When users first open the application, they will arrive at the landing page.

2. After clicking the "Get Started" button, user have to option to login/signup or continue without logging in.

3. User can enter news text in the text field and also fill additional fields for detailed classification results.

4. After clicking "Check That Fact" button, users will be able to view classification results.
